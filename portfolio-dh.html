What is/was datahub?

A SharePoint website + Bootstrap 4

The requirement was to build something that looks and feels like a website, but in a corp. environment that used the Microsoft stack. This was build in/around 2017.

  • Basic goal: A SharePoint website that doesn't look/feels like SharePoint*cough*sharepoint ribbon*cough*
  • This website was built with Bootstrap 4 and jQuery, on top of MS Sharepoint 2016.
  • Basic masterpage templates were created and allows for content managers to add pages using SP without having to fiddle around with the codes too much.
  • Zero budget :) Side-desk kind of activity

Description

The requirement was to build something that looks and feels like a website, but in a corp. environment that used the Microsoft stack.

This website was built with Bootstrap 4 and jQuery, on top of MS Sharepoint 2016. Basic masterpage templates were created and allows for content managers to add pages using SP without having to fiddle around with the codes too much.

Blurred things out in this video capture, but it essentially demonstrates that a responsive intranet can be built within SharePoint.


Final background is a picture of the castle at Carcassone that I took in 2017.

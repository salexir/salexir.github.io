Toronto Thinks: A website for a case-competition

  1. Home
  2. Toronto Thinks
 

Another responsive website

Toronto Thinks is (was?) a global health case-competition.

A challenge with a global health focus is set to delegates, who will have to conceptualise, and propose a solution to the problem set.
Borrowed from the successful models out of business schools, participants are pitted against each other to develop a solution to an assigned case.


This project was the first of its kind for undergraduates in Canada. I built this website for the 2014 iteration for several purposes:

  • to get the word out
  • act as the main information portal
  • to handle user registrations
This was the first web-design project of mine that actually had a purpose. It is definitely choppy.
